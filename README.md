Calculator App
Overview
The Calculator App is a simple web-based application developed using HTML, CSS, and JavaScript. It allows users to perform basic arithmetic calculations through an intuitive and responsive interface.

Features
Addition (+)
Subtraction (-)
Multiplication (×)
Division (÷)
Clear display functionality
Responsive user interface
Real-time calculation display
Technologies Used
HTML5
CSS3
JavaScript

Installation
Clone the Repository
git clone https://github.com/lihle-Tuta2/Calculator.git
Navigate to the Project Directory
cd Calculator
Run the Application
Open the caltulator.html file in your preferred web browser.

Usage
Open the calculator application.
Click the number buttons to enter values.
Select an arithmetic operator.
Enter the second value.
Press the equals (=) button to calculate the result.
Use the clear button to reset the calculator.
